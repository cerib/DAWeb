Resolução dos tpc 4. Nestes TPC foi-nos pedido para dividir o ficheiro dos arqueossítios em vários ficheiros xml, um ficheiro para cada arquelem. O professor acabou por disponibilizar a versão dele, o que me levou a abandonar a minha e a modificar o projecto para ser compatível com a versão do professor, no entanto no histórico de commits ainda é possível aceder à minha versão

A parte principal do TPC foi construir um servidor de ficheiros xml básico em Node.js - e os ficheiros xml devem ser convertidos para html antes do utilizador o visualizar. Sendo assim o servidor precisa de ter carregado o ficheiro xml2html.xsl antes de enviar os ficheiros xml para o browser do utilizador.

Quanto ao funcionamento, como a leitura de ficheiros não é síncrona, decidi usar Promises para garantir que o ficheiro xsl estava carregado antes de o servidor começar a ouvir a porta 7777
