Resolução do tpc 6 - neste TPC foi pedido para criarmos um server e fazer o CRUD de entradas numa "BD" em JSON de arquivos sonoros

Utilizei Express e Pug para fazer o exercício. Todos os arquivos existentes são mostrados na página inicial, juntamente com o formulário de submissão de um novo arquivo. Para visualizar ou editar um arquivo, basta clicar no link presente na linha da tabela, onde este se encontra. Utilizo o módulo uniqid para criar IDs únicos, já que não existe nenhum campo no XML original que permita identificar unicamente cada arquivo.

Para a conversão de XML para JSON utilizei uma ferramenta online. Depois corri o script em Node.js "prepara_json.js" para colocar um ID único em cada arquivo, para que o meu servidor funcionasse correctamente com os arquivos pré existentes. No caso da inserção de um novo arquivo, o id é colocado de forma transparente para o utilizador.
