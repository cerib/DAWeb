Resolução do tpc 7 - neste TPC foi pedido para criarmos um server de filmes e fazer o CRUD de filmes numa BD usando mongoDB e o módulo mongoose

Utilizei Express e Jade (Pug) para fazer o exercício. Todos os filmes existentes são mostrados na página inicial, juntamente com o formulário de submissão de um novo filme. Para visualizar ou editar um filme, basta clicar no link presente na linha da tabela, onde este se encontra.

O comportamento deste servidor é muito parecido ao do TP6, com a diferença de estar a ser usado o mongodb.
