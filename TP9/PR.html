Resolução do tpc 9 - neste TPC foi pedido para ser feito em javascript o "mongoimport" do mongo. Para executar o programa é só fazer, no mesmo directório do programa, node ./mongoimport.js -d nome_bd -c nome_collection ./caminho_do_ficheiro_json. Nesta implementação não é necessário indicar se o ficheiro é um array JSON com o comando "--jsonArray", essa detecção é feita automaticamente.

Importante: é necessário instalar os módulos jsonfile e mongoose
