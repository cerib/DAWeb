Igualdade matemática

Dois objectos matemáticos são iguais se e só se são precisamente o mesmo em todo o caminho. Isto define uma relação binária, igualdade, denotada pelo sinal de igualdade "=" em tal modo que a sentença "x = y" significa que x e y são iguais.

Algumas propriedades

  • Propriedade da substituição

    Para qualquer quantidade a e b e qualquer expressão F(x), se a = b, então F(a) = F(b).

  • Propriedade reflexiva

    Para qualquer a, a = a

  • Propriedade simétrica

    Para qualquer a e b, se a = b então b = a

  • Propriedade transitiva

    Para qualquer a, b e c, se a = b e b = c então a = c
